How to Run

The app can be running locally after cloning the repo from Github

git clone https://github.com/nezaYSR/movie-graphql.git

Running via CLI

Add git remote locally

git remote add {{remote}} https://github.com/nezaYSR/movie-graphql.git

Pull the latest of branch main (assuming there's no development/sit/uat branch)

git pull {{remote}} main

If you want to run the development server, you can find the other scripts in the package.json file located in the root folder.

npm run dev

Docker and Docker Compose

Make sure that Docker and Docker Compose are installed on your machine.

docker -v

Docker version 20.10.14, build a224086

docker-compose -v

docker-compose version 1.29.2, build 5becea4c

Then, you can run the docker-compose command or easily click the toggle button from the Docker Desktop app.

docker-compose up -d

Make sure that no other application is running on port 3000 and 5432. If it is, you can manually change the EXPOSE port in the Dockerfile located in the root folder.

netstat -a -n -o|find '3000'

Running Unit Test

npm run test
