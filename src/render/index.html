Getting Started

Welcome to the Movie GraphQL! Here, you can store movies, actors, and authors. You can view the list of movies, actors, and authors without logging in. However, to see their details, you will need to log in.

You can create a user account using an admin account. For demonstration purposes, here are the credentials:

username: "admin", password: "admin"

When registering a new user as an administrator, you should provide a username for the user. This username will allow the user to access their inbox and view the generated password that has been assigned to them. The generated password is only temporary and will be used for the user's initial login.

List of Role Privileges:

  • admin

    • Registering and Assigning Roles to New Users, Creating, Reading, Updating, and Deleting Movies, Actors, and Authors.
  • admin-support

    • Creating, Reading, Updating, and Deleting Movies, Actors, and Authors.
  • user

    • Reading Movies, Actors, and Authors details.
